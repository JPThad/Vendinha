**-Projeto:** Vendinha API.
**- O que é?:** API para gerenciamento de clientes e dívidas.
-**Tecnologias utilizadas:** ASP.NET Core; C#; Entity Framework Core; PostgreSQL; Swagger.



**-Como executar?**

**1.** Instale o PostgreSQL

**2.** Crie um banco chamado VendinhaDb

**3.** Execute o arquivo banco.sql

**4.** Configure a connection string no appsettings.json

**5.** Execute o projeto

**6.** Abra o Swagger



**-Algumas funcionalidades:** CRUD de clientes, CRUD de dívidas, paginação, filtro por nome, controle de pendências.
